# carbotrackr-react-native
# Network Error Issues?
1. Go to shared folder > api.ts
2. Change the BASE_URL to your machine's local IP address (e.g., http://192.168.x.x:3000) instead of localhost

# Issue with loading data?
1. Make sure that the MOCK_PROFILE_ID is the same ID generated by your backend/database (Neon Postgres)
2. Check every data that is related to a user profile (AddBloodPressureForm)
